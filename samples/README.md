# Sample Project Documents

Filled-in examples of the documents a project manager produces, so you can see what each one actually looks like — not just its definition.

**Everything here is fictional and for study only.** All nine documents describe the **same imaginary project**, so they cross-reference each other (the risk in the register shows up in the status report, the change in the log points back to the charter, the RACI names match throughout).

## The shared example project

**Northwind Retail — Customer Self-Service Portal** (`NWR-2026-014`)

A web portal that lets customers view orders, track shipments, start returns, and reset passwords — built to cut call-center load and lift customer satisfaction.

- **Sponsor:** Maria Alvarez (VP Customer Experience)
- **Project Manager:** Priya Shah
- **Budget:** $180,000 · **Timeline:** Jan–Jun 2026 · 2-week sprints
- **Core team:** Tom Reyes (Business Analyst), Nina Brooks (Architect), Jordan Lee (Lead Dev), Aiko Kim (QA Lead), Sam Patel (DevOps), David Cho (IT Director)

## The documents

| # | Document | Format | What it shows |
|---|---|---|---|
| 01 | [Business Case](01-business-case.md) | Markdown | Whether the project should proceed — options, costs, ROI, recommendation. |
| 02 | [Project Charter](02-project-charter.md) | Markdown | Formal authorization — objectives, success criteria, scope summary, PM authority. |
| 03 | [Scope Statement](03-scope-statement.md) | Markdown | Product vs. project scope, deliverables, acceptance criteria, exclusions. |
| 04 | [Change Management Plan](04-change-management-plan.md) | Markdown | How changes are requested, assessed, and approved. |
| 05 | [Status Report](05-status-report.md) | Markdown | A weekly progress snapshot with health, metrics, and escalations. |
| 06 | [Risk Register](06-risk-register.xlsx) | Excel | Logged risks with probability, impact, score, response, and owner. |
| 07 | [Issue Log](07-issue-log.xlsx) | Excel | Active issues with severity, owner, status, and resolution. |
| 08 | [Change Log](08-change-log.xlsx) | Excel | Submitted change requests and their CCB decisions. |
| 09 | [RACI Matrix](09-raci-matrix.xlsx) | Excel | Who is Responsible, Accountable, Consulted, and Informed for each activity. |

> Markdown files preview right here on GitHub. The Excel files (registers and logs) download to open in Excel, Google Sheets, or Numbers — that's how these live in the real world.
