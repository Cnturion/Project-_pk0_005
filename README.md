# CompTIA Project+ (PK0-005) Notes

Personal study notes for the **CompTIA Project+ PK0-005** certification, organized from the [CBT Nuggets course by Simona Millham](https://learn.adept.at/cbtnuggets/comptia-project-pk0-005).

The notes are organized **by exam domain and objective** so you can study and self-assess against the official blueprint. Written for quick scanning: short sections, bullets, and diagrams.

## Exam at a glance

- **Exam number:** PK0-005
- **Length:** 90 minutes
- **Questions:** Up to 90 (multiple-choice and performance-based)
- **Recommended experience:** 6–12 months managing projects in an IT environment

## How to use these notes

Pick a domain, work through its objectives, then check yourself against the official objectives PDF. Use the formulas PDF for the Critical Path Method and Earned Value Management math.
